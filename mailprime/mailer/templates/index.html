So what is MailPrime?

MailPrime is a web-based email campaign tool that gives users the ability to create, schedule, deploy, and track large email campaigns.

MailPrime is a comprehensive email-service that gives users a selection of great-looking HTML email message templates. MailPrime sends the HTML formatted message along with a more universal plain-text version as well.

MailPrime also implements an opt-in model for recipients. Possible recipients will need to authorize their participation before recieving messages from the campaign administrator. Users are able to opt-out of the campaign at any time. If a user does opt-out of the campaign, all personal and tracking data belonging to them will be promtly deleted.

In addition, never miss a message with the MailPrime's message-scheduler. MailPrime's cron-based scheduler allows administrators to choose when to deploy their campaign messages, aligning with the needs of the campaign administrator.

MailPrime also displays tracking data back to the administrator too. Visualization tools are used to create reports that show recipient participation.
